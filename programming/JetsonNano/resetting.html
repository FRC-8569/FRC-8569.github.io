Preparing For the Resetting

Everything you need

  • Jetson Orin Nano
  • SSD
  • ubuntu computer
  • jumper
  • USB C cable
  • Power Supply
  • DP wire (not required)
  • SDK Manager(software)
    •

The way to download the software

Go the website and choose the choice which fits you (here is .deb ubuntu) and follows the instructions of the website and done!

Don't forget to...

use a jumper to connect pin 9 and 10 at J14 (SYS RST and GND)

connect Jetson Nano to computer by USB C port at J5

Let's Started

1. Open the SDK manager and login to you NVIDIA developer account

2. Choose Jetson from the top instead Data Science, it will choose the device automatically if detected

3. In the next page, you can install the additional software like CUDA and TensorRT, after choosing, click the next step to start installation

4. then you can take a nap for waiting it. (it will pop up some configuration page while installation)

✅ after the installation, you're reset the Jetson Orin Nano successfully
